# git commit 注释

feat: 新功能\
fix: 修补 bug\
docs: 文档\
style: 格式（不影响代码运行的变动）\
refactor: 重构 \
perf: 改善性能 \
test: 测试 \
build: 变更项目构建或外部依赖（例如 scopes: webpack、gulp、npm 等）\
ci: 更改持续集成软件的配置文件和 package 中的 scripts 命令，例如 scopes: Travis, Circle 等 \
chore: 变更构建流程或辅助工具 \
revert: 代码回退
